# Verify Development Environment

Run the following commands.

---

## Git

git --version

---

## AWS CLI

aws --version

---

## Terraform

terraform version

---

## Docker

docker --version

---

## kubectl

kubectl version --client

---

## Helm

helm version

---

## Databricks CLI Setup

Adicionar ao Linux:
```bash
pip install databricks-cli
```
Verificar:
```bash
databricks --version
```
