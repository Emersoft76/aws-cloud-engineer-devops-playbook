# Windows Development Environment Setup

This guide installs the tools required for working with AWS cloud engineering projects.

---

## Required Tools

- Git
- AWS CLI
- Terraform
- Terragrunt
- Docker
- kubectl
- Helm
- Python
- Node.js

---

## Install Git

Download:

https://git-scm.com/download/win

Verify:

git --version

---

## Install AWS CLI

Download:

https://aws.amazon.com/cli/

Verify:

aws --version

---

## Install Terraform

Download:

https://developer.hashicorp.com/terraform/downloads

Verify:

terraform version

---

## Install Docker Desktop

Download:

https://www.docker.com/products/docker-desktop/

Verify:

docker --version
