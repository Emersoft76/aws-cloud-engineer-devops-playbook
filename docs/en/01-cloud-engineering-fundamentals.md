# Cloud Engineering Fundamentals

Cloud engineering involves designing and operating scalable cloud infrastructure.

Key responsibilities include:

- provisioning infrastructure
- networking configuration
- security management
- automation

---

## Core AWS Services

### Compute

- Amazon EC2
- AWS Lambda
- Amazon ECS
- Amazon EKS

### Storage

- Amazon S3
- Amazon EBS
- Amazon EFS

### Networking

- Amazon VPC
- NAT Gateways
- Internet Gateways
- Route Tables
