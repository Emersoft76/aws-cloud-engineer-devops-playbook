# AWS Cloud Engineering & DevOps Playbook

Author: Emerson Maciel, Cloud Architect | AWS

---

## Overview

This repository provides a comprehensive guide to building and operating **enterprise-grade cloud platforms on AWS**.

It covers the core domains required by modern cloud engineering teams:

- Cloud Infrastructure
- DevOps Automation
- Infrastructure as Code
- CI/CD pipelines
- Container orchestration
- Observability
- Data platforms
- Networking and CDN architectures

---

## Target Audience

This repository is designed for:

- Cloud Engineers
- DevOps Engineers
- Platform Engineers
- Site Reliability Engineers
- Cloud Architects

---

## What You Will Learn

- How to design AWS infrastructure
- How to automate infrastructure using Terraform
- How to build CI/CD pipelines
- How to run Kubernetes in production
- How to implement observability
- How to design enterprise cloud platforms

---

## Repository Philosophy

This repository focuses on **real production architectures** used by enterprise cloud teams.
