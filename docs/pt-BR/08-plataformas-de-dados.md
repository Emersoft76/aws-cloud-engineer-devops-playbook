# Plataformas de Dados

Empresas modernas utilizam plataformas de dados para armazenar e analisar grandes volumes de informação.

---

## Data Lake

Repositório centralizado para armazenamento de dados.

Na AWS normalmente utilizamos:

Amazon S3

---

## Databricks

Plataforma de analytics baseada em Apache Spark.

https://databricks.com

Permite:

- processamento de dados
- machine learning
- análise em larga escala

---

## Arquitetura Medallion

Estratégia comum de organização de dados.

### Bronze

Dados brutos.

### Silver

Dados limpos e transformados.

### Gold

Dados prontos para consumo analítico.
