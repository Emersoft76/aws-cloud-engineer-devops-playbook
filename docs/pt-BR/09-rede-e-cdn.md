# Redes e CDN

Arquiteturas cloud dependem de redes bem projetadas para garantir segurança e desempenho.

---

## VPC (Virtual Private Cloud)

Rede isolada dentro da AWS.

Componentes principais:

- subnets
- route tables
- internet gateways
- NAT gateways

---

## Balanceamento de carga

Distribui tráfego entre múltiplos servidores.

Na AWS usamos:

Application Load Balancer (ALB)

---

## CDN (Content Delivery Network)

CDNs distribuem conteúdo globalmente através de servidores de borda.

Benefícios:

- baixa latência
- alta disponibilidade
- redução de carga no servidor

---
